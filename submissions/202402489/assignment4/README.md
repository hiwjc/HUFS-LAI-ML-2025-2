# [데이터 분석 보고서]
- 처음 주제로 잡았던 '강의평 기반 추천 시스템'은 유의미한 학습을 시킬 수 있는 양의 데이터를 에브리타임상 크롤링하는 데에 보안이나 윤리적 문제 우려되는 부분이 있어 주제를 변경하였습니다. 변경된 주제를 지난 과제에 수정해 놓았습니다.

## 1. 분석 개요

분석 대상: 직장인 리뷰 데이터 5,599건

목적: 직무와 기업별 만족도 결정 요인 파악 및 기업 유형 분류

분석 기법: OLS 회귀분석, PCA(주성분분석), K-Means 군집화, NLP(형태소 분석)

## 2. 상세 분석 결과
# 1) 만족도 결정 요인 분석 (다중회귀분석)

- 분석 결과: 모델 설명력(R-squared) 0.735로 매우 높음.

- 영향력 순위 (회귀계수 기준):

   1위: 경영진 (0.3087) - 압도적 영향력

   2위: 사내문화 (0.2302)

   3위: 승진 기회 (0.1998)

   4위: 복지 및 급여 (0.1738)

   5위: 업무와 삶의 균형 (0.1137)

- 시사점:

워라밸은 종합 만족도를 높이는 데 기여도가 가장 낮음 (기본 요건).

경영진 점수가 1점 오르는 것이 워라밸 점수 1점 오르는 것보다 만족도 상승 효과가 약 2.7배 큼.

# 2) 데이터 구조 및 패턴 분석 (PCA)

- 제1주성분 (PC1, 설명력 70.9%):

  모든 평가 항목이 양의 방향을 가리킴.

  좋은 회사는 모든 면에서 좋고, 나쁜 회사는 모든 면에서 나쁜 '빈익빈 부익부' 현상이 뚜렷함.

- 제2주성분 (PC2, 설명력 10.0%):

  '워라밸' vs '승진기회/경영진'의 대립 구조.

  워라밸이 아주 높은 회사는 상대적으로 승진 기회나 리더십의 추진력이 약한 경향이 있음.

  # 3) 기업 유형 분류 (K-Means Clustering)

- Cluster 1 (좋은 직장 - 이상적):

  종합평점: 4.32점

  특징: 급여, 워라밸, 문화 등 모든 지표가 최상위이나 존재 비율이 낮음.

- Cluster 2 (워라밸 안식처 - 안정적):

  종합평점: 3.35점

  특징: 워라밸(3.44)은 준수하나 경영진/비전(2.83) 점수가 상대적으로 낮음.

- Cluster 3 (정체된 조직 - 위험):

  종합평점: 2.55점

  특징: 모든 점수가 2점대 중반으로 큰 장점도 단점도 없으나 활력이 부족함.

- Cluster 0 (기피 조직 - 매우 위험):

  종합평점: 1.45점

  특징: 경영진 점수(1.23)가 최악이며 문화와 시스템이 붕괴된 상태.
