# 5th Assignmen : 프로젝트 결과 및 파이프라인 요약

## 프로젝트 개요: [AI 기반 기업 리뷰 분석 및 하이브리드 추천 시스템]

본 프로젝트는 기업의 비정형 텍스트 리뷰 데이터(장/단점, 요약)와 정형 평점 데이터(복지, 워라밸 등 6개 항목)를 통합 분석하여, 
1) 기업별 핵심 인사이트를 도출하고
2) 사용자의 자연어 희망 직무환경을 이해하는 맞춤형 기업 추천 시스템을 구축하는 것이 목표

### 전체 프로젝트 구성

해당 프로젝트의 과정은 모델 학습과 평가, 서비스 적용(Inference)의 3단계 파이프라인으로 구성

 # AI 모델 학습 (Model Training)
 - 텍스트의 '맥락'을 이해하고 점수화할 수 있는 **지능(Brain)**을 만드는 과정
 
 - 데이터 분할 (Data Split)Train (80%): 모델이 학습.
 
 - Validation (10%): 과적합 방지, Best Model 저장용
 
 - Test (10%): 학습 때 전혀 보지 못한 데이터로 객관적 성능 평가.
 
 - 모델 아키텍처 (KLUE-BERT + Regression Head): 입력으로 리뷰 텍스트 전체 ("연봉은 적지만 수평적이에요"), 출력으로 6가지 세부 평점 예측값이 출력되는 모델 (Vector Size: 6)
 
 - 학습 원리: 모델은 수천 개의 리뷰를 읽으며 단어와 평점 간의 상관관계를 스스로 학습.
 
 - "꼰대" 등장 $\rightarrow$ 사내문화 점수 하락
 
 - "야근" 등장 $\rightarrow$ 워라밸 점수 하락

 - 학습 결과 및 최선의 체크포인트 저장

# 추천 시스템 구현 (Inference & Recommendation)

학습된 모델을 활용해 실제 사용자의 질문에 답하는 서비스 단계

임베딩(Embedding) 생성

- 학습된 모델(.pt)을 통해 원본 엑셀 데이터의 모든 리뷰 텍스트를 **벡터(숫자 배열)**로 변환

- 이 벡터에는 단순 단어 매칭이 아닌, **'리뷰의 뉘앙스와 평점 정보'**가 압축되어 있다고 볼 수 있음. (평점과 텍스트 의미의 대응을 학습하면서)

- 코사인 유사도 검색 (Vector Search)

- 사용자 입력: "돈 많이 주고 성장하는 곳"

- 매칭 원리: 사용자 입력 문장의 벡터와 가장 각도(방향)가 비슷한 기업 리뷰 벡터를 찾음

- 결과: 키워드가 정확히 일치하지 않아도, 의미적으로 가장 유사한(급여↑, 승진↑) 기업을 추천

- 이 시스템은 향후 채용 플랫폼의 맞춤형 공고 추천이나 사내 조직문화 진단 솔루션으로 확장 가능
